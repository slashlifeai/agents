DID Webs

Decentralized Identifiers (DIDs) are a new type of identifier that enables verifiable, decentralized digital identity. The `did:web` method leverages existing web infrastructure to create and resolve DIDs. To resolve a `did:web` DID, you can construct a URL by replacing `did:web:` with `https://`, and then appending `/.well-known//did.json`. For example, the DID `did:web:agents.slashlife.ai:max` resolves to `https://agents.slashlife.ai/.well-known/max/did.json`.
